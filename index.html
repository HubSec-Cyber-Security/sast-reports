Anahtar Çıkarımlar

KonuBulgu
Lockfile bağımsızlığıjuice-shop'ta Trivy 0 bulgu üretirken Aqua 79 CVE buldu — Aqua package.json range'lerinden çözümleme yapıyor. Trivy için npm install + lockfile zorunlu.
Veri kaynağı genişliğiAqua, NSWG (Node Security WG) ve GHSA gibi non-CVE advisory'leri de tarıyor. NodeGoat'ta NSWG-ECO-118 / 28; WebGoat'ta GHSA-72hv-8253-57qq sadece Aqua'da.
Eski CVE kapsamaAqua, 2014–2018 arası eski CVE'leri (qs, mime, async, braces, tar, decode-uri-component) Trivy'den daha iyi yakalıyor — uzun kuyruk advisory feed'i daha geniş.
Yeni CVE hız (2026)Trivy bazı 2025/2026 CVE'lerini (CVE-2026-40477/40478/41901 thymeleaf, CVE-2026-27601 underscore) Aqua'dan önce yakalamış. Aqua DB güncellemesinin sonrası beklenen.
Severity scoringNodeGoat'taki 8 CVE'de Aqua daha agresif: y18n / ini / set-value HIGH→CRITICAL; express ekosistemini LOW→MEDIUM. Aqua kendi triage skoru uyguluyor, Trivy NVD CVSS'e sadık.
Triage zenginliğiAqua'da EPSS, Direct/Transitive, Package Roots, Vendor Fix, Reachability alanları var. Örn. WebGoat'ta xstream CVE-2021-39144 EPSS=0.94 — aktif sömürü riski yüksek, öncelikli aksiyon.
Pratik öneriİki aracı tamamlayıcı kullan: Aqua → öncelik / triage / iş akışı; Trivy → CI/CD'de hızlı geri bildirim ve bağımsız doğrulama. Tek araca güvenmek gözden kaçırma riski yaratır.
